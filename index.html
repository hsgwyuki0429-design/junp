足場の色と能力

青 / ノーマル
普通の足場。安全に着地できる基本ブロック。
緑 / ムービング
左右にゆっくり動く。乗ると一緒に運ばれる。
赤 / バニッシュ
一定周期で点滅して消える。消える前に飛び移ろう。
黄 / ブースター
乗った瞬間に大ジャンプ！一気に上へ飛べる。
紫 / クランブル
乗ると崩れ落ちる。素早く次へ渡ろう（時間で復活）。
水色 / コンベア
ベルトのように片方向へ流される。矢印が向き。
氷 / アイス
ツルツル滑って止まりにくい。慣性に注意。
黄緑 / スライム
ポヨンと弾む。ブースターより控えめな連続ジャンプ。
ピンク / トグル
周期で実体化⇔すり抜けを切替。光っている時だけ乗れる。
空色 / ウィンド
上空に上昇気流。上を通ると押し上げられ滞空できる。
赤発光 / 重力異常 (ハード)
乗ると重力が弱まりフワッと浮く。ハードモード限定。
